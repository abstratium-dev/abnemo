For Docker enrichment (🐳 🐋), run this on your remote server:

# Download and run the export script:
curl -O https://raw.githubusercontent.com/YOUR_REPO/abnemo/main/export_docker_info.sh
chmod +x export_docker_info.sh
./export_docker_info.sh > docker_and_iptables.txt
sudo iptables -L -v -n >> docker_and_iptables.txt
# Then paste the contents of docker_and_iptables.txt below

Or just paste: sudo iptables -L -v -n (without Docker enrichment)

Legend

✅ Allow - Traffic is accepted
❌ Block - Traffic is dropped/rejected
🐳 Container - Docker container
🐋 Network - Docker network
🏠 Private IP - Internal network
🔁 Loopback - Local traffic
